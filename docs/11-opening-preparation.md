# 開所前準備

開所日を前に、事業所の物理的・制度的な準備を完了させてください。初日にスムーズに利用者を迎えられる状態を目指します。

## 備品設置

- [ ] 作業机・椅子の配置
- [ ] 相談室の家具・備品
- [ ] 事務デスク・PC・モニター
- [ ] 書庫・ファイルボックス
- [ ] 掲示板・ホワイトボード
- [ ] 救急箱・AED（該当する場合）
- [ ] 消耗品（文房具・コピー用紙等）の配置

## 送迎体制

利用者送迎を行う場合、開所前に車両と運行体制を整えてください。

- [ ] 送迎用車両の確保（リース・購入・既存車活用など）→ **上位店へ確認**
- [ ] 車両の任意保険加入・点検・車検の確認
- [ ] 送迎を担当する職員の確保・運転可否の確認
- [ ] 送迎ルート・時間帯のたたき台作成
- [ ] 車内の安全装備（シートベルト等）の確認

::: tip
送迎の実施有無・車両の手配方法は、自治体の基準や上位店の方針により異なります。独断で決めず、早い段階から上位店へ確認してください。
:::

## 職員の昼食手配

開所後、職員が事業所で昼食をとる場合は、弁当手配業者を事前に決めておいてください。

- [ ] 弁当手配業者の候補をリストアップ・比較
- [ ] 業者への見積・試食・配達条件の確認
- [ ] 注文方法・締切時間・配達時間の確定
- [ ] アレルギー等への対応可否を確認
- [ ] 昼食費の負担（事業所負担・職員負担等）を決定 → **上位店へ確認**
- [ ] 注文・受け取りの担当とルールを職員に共有

## PC・IT環境

- [ ] PC設定完了（全職員分）
- [ ] Wi-Fi接続・速度確認
- [ ] プリンター設置・動作確認
- [ ] メールアカウントの動作確認
- [ ] 共有ドライブ・フォルダの整備
- [ ] バックアップ体制の確認

## 書類・ファイル整備

- [ ] 利用者ファイルのフォーマット準備
- [ ] 記録用紙・帳票の準備
- [ ] 契約書類のテンプレート
- [ ] 掲示物（利用者向け・職員向け）
- [ ] 個人情報保護・利用規約等の掲示
- [ ] 緊急連絡先リストの掲示

## 職員研修

- [ ] 事業所理念・運営方針の共有
- [ ] 利用者支援の基本方針
- [ ] 作業マニュアル・検品ルールの説明
- [ ] 個別支援計画の作成フロー
- [ ] 記録の取り方
- [ ] 緊急時対応・事故発生時の手順
- [ ] 初日の役割分担・スケジュール確認

## 緊急時対応

- [ ] 119・110の連絡手順を確認
- [ ] 緊急連絡先リストの配布
- [ ] 火災・地震等の避難経路確認
- [ ] 利用者の急病時対応フロー確認
- [ ] 管理者・サビ管の連絡体制確認

## 清掃・環境

- [ ] 全エリアの清掃完了
- [ ] 作業室・相談室・トイレの清掃
- [ ] 換気・空調の確認
- [ ] ゴミ捨て場・分別の確認

## 掲示物

- [ ] 事業所名称・営業時間
- [ ] 利用者向け案内
- [ ] 個人情報保護方針
- [ ] 苦情・相談窓口
- [ ] 緊急連絡先
- [ ] 避難経路図（該当する場合）

## 利用者受入導線

- [ ] 入口から受付までの導線確認
- [ ] 見学時の案内ルート確認
- [ ] 体験利用時の動線確認
- [ ] トイレ・休憩スペースの案内
- [ ] 初回来所時の受付フロー確認

## 初日の流れ

| 時間 | 内容 | 担当 |
|------|------|------|
| 開所前 | 全職員集合・最終ブリーフィング | 管理者 |
| 開所 | 利用者受入開始 | 全職員 |
| 午前 | 利用者対応・作業開始 | 生援・職指 |
| 昼 | 休憩・記録 | 全職員 |
| 午後 | 利用者対応・作業継続 | 生援・職指 |
| 閉所後 | 振り返り・記録確認 | 管理者・サビ管 |

- [ ] 初日スケジュールを全職員に共有
- [ ] 各時間帯の担当者を明確化
- [ ] 想定問い合わせ・トラブルへの対応を確認

## 開所前最終確認

開所7日前〜前日に、[開所前最終チェックリスト](/12-checklist) および [テンプレート版](/templates/opening-final-checklist) を使って最終確認を行ってください。

::: tip
前日は全職員で最終確認を行い、不明点があればその場で解決してください。
:::
