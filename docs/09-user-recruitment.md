# 利用者募集準備

B型事業所は、開所後すぐに利用者が集まるわけではありません。**開所前から**利用者募集の準備と営業活動を開始してください。

## なぜ開所前から必要か

- 利用者の受入れには、見学・体験・相談・契約等のプロセスが必要
- 相談支援事業所・病院等への案内には時間がかかる
- 開所直後から利用者が来るよう、早めの営業が重要

## 準備項目

### 見学・体験の受付体制

- [ ] 見学受付の方法（電話・Web・LINE等）を決定
- [ ] 見学可能日時の設定
- [ ] 体験利用の受付フロー作成
- [ ] 見学時の案内内容・資料の準備
- [ ] 見学記録のフォーマット準備 → [見学対応記録](/templates/visit-record)
- [ ] 体験利用記録のフォーマット準備

### 営業・案内活動

- [ ] 相談支援事業所への営業・案内
- [ ] 病院・クリニックへの案内
- [ ] 支援学校への案内
- [ ] 地域包括支援センター・市区町村窓口への案内
- [ ] パンフレットの作成・配布

### 問い合わせ対応

- [ ] 問い合わせ受付方法の整備
- [ ] 問い合わせ対応フローの作成
- [ ] 問い合わせ記録のフォーマット → [問い合わせシート](/templates/user-inquiry-sheet)
- [ ] 初回対応の担当者・返答期限の設定

## 営業先

| 営業先 | 案内方法 | 備考 |
|--------|---------|------|
| 相談支援事業所 | 訪問・資料持参 | 最も重要な送客元 |
| 精神科病院 | 訪問・資料置き | 退院前後の利用者 |
| 心療内科 | 訪問・資料置き | 通院中の利用者 |
| 地域活動支援センター | 訪問・連携 | 地域の支援拠点 |
| 特別支援学校 | 訪問・説明会 | 卒業後の利用者 |
| 市区町村窓口 | 登録・情報提供 | 地域の窓口 |

## 見学・体験の流れ

```
問い合わせ → 初回対応 → 見学予約 → 見学実施 → 体験利用 → 利用申込 → 契約
```

### 各ステップのポイント

| ステップ | ポイント |
|---------|---------|
| 問い合わせ | 24時間以内に返答、記録を残す |
| 見学 | 事業所の雰囲気・作業内容を具体的に説明 |
| 体験利用 | 1〜5日程度、実際の作業を体験 |
| 利用申込 | 必要書類の案内、サビ管との面談 |

## 注意点

- 利用者の受入れ基準・定員については上位店の指示に従ってください → **上位店へ確認**
- 体験利用の実施方法（日数・内容等）も上位店と確認してください
- 個人情報の取り扱いには十分注意してください
- 過度な勧誘は避け、利用者本人の意思を尊重してください
