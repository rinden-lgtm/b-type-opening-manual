# IT・集客環境整備

開所前に、利用者や関係機関から事業所を見つけられ、問い合わせできる環境を整えてください。

## 整備項目一覧

| 項目 | 優先度 | 目安時期 |
|------|--------|---------|
| ホームページ | 高 | 60日前〜 |
| Googleビジネスプロフィール | 高 | 30日前〜 |
| LINE公式アカウント | 高 | 30日前〜 |
| Google Workspace | 高 | 30日前〜 |
| メールアドレス | 高 | 30日前〜 |
| 電話番号 | 高 | 60日前〜 |
| 問い合わせフォーム | 高 | 30日前〜 |
| パンフレット | 中 | 30日前〜 |
| 名刺 | 中 | 14日前〜 |
| Instagram等のSNS | 中 | 30日前〜 |

## 各項目の準備手順

### ホームページ

事業所の顔となるWebサイトです。開所前に公開できる状態にしてください。

- [ ] 事業所名・所在地・連絡先の掲載
- [ ] サービス内容・支援方針の説明
- [ ] 見学・体験の案内
- [ ] 問い合わせフォームまたは連絡先
- [ ] アクセス・地図の掲載
- [ ] スマートフォン対応（レスポンシブ）

### Googleビジネスプロフィール

Googleマップ・検索に事業所情報を登録します。

- [ ] Googleビジネスプロフィールのアカウント作成
- [ ] 事業所名・住所・電話番号・営業時間の登録
- [ ] 外観・内観写真の登録
- [ ] サービス内容の記載
- [ ] 開所後、利用者からの口コミ対応を促す

### LINE公式アカウント

問い合わせ・見学予約の受付チャネルとして活用します。

- [ ] LINE公式アカウントの開設
- [ ] プロフィール・挨拶メッセージの設定
- [ ] 見学予約・問い合わせの受付方法
- [ ] パンフレット・HPへのリンク設定

### Google Workspace

メール・カレンダー・ドライブ等の業務基盤です。

- [ ] Google Workspaceアカウントの取得
- [ ] 事業所用メールアドレスの作成（例：info@事業所用ドメイン）
- [ ] 職員アカウントの作成
- [ ] 共有ドライブ・カレンダーの設定

### 電話番号

- [ ] 固定電話または050番号の取得
- [ ] 留守番電話・転送の設定
- [ ] HP・Googleビジネスプロフィール・名刺への掲載

### 問い合わせフォーム

- [ ] HPに問い合わせフォームを設置
- [ ] 通知先メールアドレスの設定
- [ ] 自動返信メッセージの設定
- [ ] 個人情報の取り扱いに関する表記

### パンフレット・名刺

- [ ] パンフレットのデザイン・印刷
- [ ] 名刺のデザイン・印刷（職員分）
- [ ] 営業先配布用の部数確保

### SNS（Instagram等）

- [ ] アカウント開設
- [ ] プロフィール・事業所紹介の設定
- [ ] 開所前から活動内容の発信開始（任意）

## チェックリスト

- [ ] ホームページを公開した
- [ ] Googleビジネスプロフィールを登録した
- [ ] LINE公式アカウントを開設した
- [ ] Google Workspace・メールアドレスを整備した
- [ ] 電話番号を取得し、各所に掲載した
- [ ] 問い合わせフォームを設置した
- [ ] パンフレット・名刺を準備した
- [ ] SNSアカウントを開設した（任意）

## 注意点

- ドメイン・メール・HP制作は上位店の指示に従ってください
- 個人情報を扱うフォームには、プライバシーポリシーを掲載してください
- 開所前からGoogleビジネスプロフィールを登録しておくと、開所後すぐに検索されやすくなります
