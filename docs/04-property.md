# 物件候補の準備と上位店への提出

物件の探索・候補資料の作成・**最終選定・決定**は**開設者側**が担当します。上位店は候補の確認と**最終選定に関するアドバイス**を提供します。

::: danger 絶対に守ること
**契約前に必ず上位店のアドバイスを受けてください。アドバイスを反映せずに物件契約をしてはいけません。最終決定は開設者が行います。**
:::

## 役割分担

| 項目 | 担当 |
|------|------|
| 物件候補の探索 | 開設者 |
| 物件候補資料の作成 | 開設者 |
| 上位店への提出 | 開設者 |
| 物件の最終選定・決定 | 開設者 |
| 最終選定に関するアドバイス | 上位店 |
| 自治体要件との適合確認 | 上位店（助言） |

## 物件候補として確認する項目

### 基本情報

- [ ] 住所
- [ ] 家賃（月額）
- [ ] 敷金・礼金
- [ ] 面積（㎡）
- [ ] 間取り

### 事業所として必要なスペース

- [ ] 作業室の確保（利用者が作業できる広さ）
- [ ] 相談室の確保（個別面談・相談ができる部屋）
- [ ] 事務スペース（職員用デスク・書類保管）
- [ ] トイレ（利用者・職員用）
- [ ] 駐車場（利用者・職員の来所手段に応じて）

### 立地・環境

- [ ] 最寄駅・バス停（アクセス）
- [ ] 周辺環境（騒音・治安・近隣施設）
- [ ] バリアフリー対応（段差・エレベーター等）

### 設備・資料

- [ ] 消防設備（消火器・非常口等）
- [ ] 写真（外観・内観・周辺）
- [ ] 図面（間取り図・平面図）

## 上位店への提出資料

候補物件ごとに、以下の資料をセットで提出してください。

| 資料 | 内容 |
|------|------|
| 物件概要 | 住所・家賃・面積・間取り・賃貸条件の一覧 |
| 間取り図 | 各部屋の用途が分かる平面図 |
| 外観写真 | 建物外観（複数角度） |
| 内観写真 | 作業室・相談室・事務スペース・トイレ等 |
| 周辺地図 | 最寄駅・バス停・周辺施設が分かる地図 |
| 賃貸条件 | 家賃・敷金・礼金・契約期間・更新条件等 |
| 懸念点 | 自分で気づいた懸念・確認したい点 |

::: tip テンプレート
[物件候補提出チェックリスト](/templates/property-checklist) を使って、提出前に漏れがないか確認してください。
:::

## 物件探索の進め方

1. **条件を整理** — 希望エリア・予算・必要面積・必須設備を明確にする
2. **候補をリストアップ** — 不動産サイト・仲介業者・直接問い合わせ等
3. **現地確認** — 必ず現地を見て、写真・図面を取得する
4. **資料を作成** — 提出資料セットを整える
5. **上位店に提出** — 候補ごとに資料を提出し、フィードバックを受ける
6. **アドバイスを踏まえて決定** — 上位店のアドバイスを参考に、**開設者が最終選定・契約を判断**

## 注意点

- 内見時に**写真・図面の撮影許可**を確認してください
- 賃貸条件（更新料・解約条件等）は必ず確認し、資料に記載してください
- 内装工事が必要な場合は、概算費用と工期も併せて整理してください
- 指定申請に必要な要件（面積・設備等）の適合は**上位店が助言**します → **上位店へ確認**
- 複数候補を同時に提出し、比較検討できるようにしてください
