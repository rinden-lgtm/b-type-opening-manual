# 作業・仕事の確保

就労継続支援B型の運営では、**利用者に提供する作業**と**工賃原資の確保**が非常に重要です。作業がなければ利用者を受け入れることができず、事業として成り立ちません。

::: warning 最重要項目
サビ管確保・物件候補と並び、**作業・仕事の確保は開設準備の三大重要項目**のひとつです。指定取得前から営業活動を開始してください。
:::

## なぜ重要か

- B型事業所は、利用者に**就労の機会**を提供するサービスです
- 利用者への**工賃支払い**には、作業受注による収入が原資となります
- 作業量が不足すると、利用定員を活かせず、事業の収支が悪化します
- 開所後すぐに利用者が来ても、提供できる作業がなければ受入れできません

## 開設前に最低限準備すべき内容

- [ ] メイン作業の決定（主力となる作業内容）
- [ ] 作業発注元の確保（1社以上、できれば2〜3社）
- [ ] 単価確認（作業単位あたりの単価）
- [ ] 月間作業量確認（月にどれだけの作業量があるか）
- [ ] 納期確認（納品期限・頻度）
- [ ] 納品方法確認（納品場所・方法・検品フロー）
- [ ] 作業マニュアル作成（利用者が迷わず作業できる手順書）
- [ ] 検品ルール作成（品質基準・不良品の扱い）
- [ ] **利用者10名程度でも回せる作業量**の確保
- [ ] 将来的に**20名程度まで増えても対応できる**作業候補の準備

## 作業例

| カテゴリ | 作業例 |
|---------|--------|
| 軽作業 | シール貼り、封入作業、検品、梱包 |
| 清掃 | オフィス清掃、店舗清掃、施設清掃 |
| 事務・PC | データ入力、EC出品、SNS投稿補助 |
| 農業 | 農作業、収穫、選別、梱包 |
| 創作 | ハンドメイド制作、簡易加工 |

::: tip 選定のポイント
- 利用者の特性（身体・精神・知的）に合った作業を選ぶ
- 単純作業から始め、段階的に難易度を上げられるものが望ましい
- 季節変動がある作業は、通年対応できる作業も併せて確保する
:::

## 営業先候補

| 業種 | 想定作業 |
|------|-----------|
| 地元企業 | 封入・梱包・検品 |
| 印刷会社 | 封入・折り・仕分け |
| EC事業者 | 出品作業・梱包・発送準備 |
| 不動産会社 | チラシ封入・データ整理 |
| 清掃会社 | 清掃業務の下請け |
| 農家 | 農作業・選別・梱包 |
| 飲食店 | 簡易調理補助・清掃 |
| 小売店 | 商品陳列・在庫整理 |

## 営業の進め方

1. **リスト作成** — [作業発注元リスト](/templates/work-client-list) に候補を記載
2. **アプローチ** — 電話・訪問・メールで商談の機会を得る
3. **商談** — [営業トークスクリプト](/templates/sales-talk-script) を参考に提案
4. **記録** — 商談内容を記録し、フォローアップ
5. **契約** — 単価・作業量・納期・納品方法を書面で確認
6. **マニュアル化** — 作業手順・検品ルールを文書化

## 必要テンプレート

| テンプレート | 用途 |
|-------------|------|
| [作業発注元リスト](/templates/work-client-list) | 営業先候補の管理 |
| [営業トークスクリプト](/templates/sales-talk-script) | 商談時の話法 |
| 作業単価表 | 発注元ごとの単価管理 |
| 作業マニュアル | 利用者向け作業手順 |
| 納品チェック表 | 納品前の品質確認 |

## 作業量の目安

| 利用者数 | 目安 |
|---------|------|
| 10名 | 月間で全員が週3日以上作業できる量 |
| 15名 | 上記＋予備作業（繁忙期・欠員対応） |
| 20名 | 複数作業・複数発注元で分散 |

::: tip 目標
開所時点で**最低10名分の作業量**を確保し、**20名規模まで拡張可能な作業候補**を2〜3種類準備しておくことを目標にしてください。
:::

## 注意点

- 作業契約の内容（単価・納期・検品基準）は書面で確認してください
- 個人情報を扱う作業は、情報管理の体制を整えてから受注してください
- 季節変動の大きい作業のみに依存しないでください
- 契約内容に法的判断が必要な場合は → **上位店へ確認**
