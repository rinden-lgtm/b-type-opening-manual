# 人員採用と勤務体制

B型事業所の運営には、法令で定められた職種の配置が必要です。サビ管と合わせて、早い段階から採用活動を開始してください。

## 必要職種

| 職種 | 主な役割 | 備考 |
|------|---------|------|
| 管理者 | 事業所全体の運営管理 | 開設者または指定者 |
| サービス管理責任者 | 個別支援計画・職員管理・サービス管理 | **法的必須** → [詳細](/03-sabikan) |
| 生活支援員 | 利用者の日常生活・就労支援 | 配置基準あり |
| 職業指導員 | 作業指導・就労支援 | 配置基準あり |

::: warning 配置基準について
職員の配置基準（人数・常勤換算等）は自治体・法令により異なります。**上位店へ確認**してください。
:::

## 採用開始時期

| 職種 | 推奨開始時期 |
|------|------------|
| サービス管理責任者 | 開設意思決定直後（最優先） |
| 生活支援員 | 90日前〜60日前 |
| 職業指導員 | 90日前〜60日前 |
| 管理者 | 60日前〜30日前 |

## 勤務表作成

- [ ] 曜日別・時間別の勤務表を作成
- [ ] サビ管の常勤配置を満たす勤務表にする → **上位店へ確認**
- [ ] 利用者利用時間帯に必要な職員が配置されている
- [ ] 欠員時の代替体制を検討
- [ ] 勤務表を職員全員に共有

## 雇用契約書

- [ ] 雇用契約書の作成（上位店の指示に従う）
- [ ] 給与・勤務時間・休日・試用期間等を明記
- [ ] 職員全員と契約締結
- [ ] 契約書の控えを保管
- [ ] 指定申請用に上位店へ提出

## 資格証確認

| 職種 | 確認する資格・証明 |
|------|------------------|
| サービス管理責任者 | サビ管研修修了証 |
| 生活支援員 | 実務者研修修了証等 |
| 職業指導員 | 職業指導員研修修了証（該当する場合） |

::: tip
資格要件は自治体・上位店の指示に従って確認してください。
:::

## 職員研修

開所前に以下の研修を実施してください。

- [ ] 事業所の理念・運営方針の共有
- [ ] 利用者支援の基本方針
- [ ] 作業内容・マニュアルの説明
- [ ] 個別支援計画の作成・更新フロー
- [ ] 記録の取り方
- [ ] 緊急時対応・事故発生時の手順
- [ ] 個人情報・情報管理
- [ ] 初日の役割分担・スケジュール

## 緊急連絡体制

- [ ] 緊急連絡先リストの作成（職員全員）
- [ ] 管理者・サビ管の連絡先を明確化
- [ ] 利用者・家族への緊急連絡フローを確認
- [ ] 119・110等の連絡手順を掲示
- [ ] 夜間・休日の対応体制を決定

## 注意点

- 雇用契約書の内容は上位店の指示に従ってください
- 配置基準を満たさない勤務表では指定取得できない場合があります → **上位店へ確認**
- 兼務の可否は職種・自治体により異なります → **上位店へ確認**
- 採用が遅れる場合は、早めに上位店に相談し、スケジュール調整を行ってください
