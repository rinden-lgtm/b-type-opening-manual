# 指定申請準備への協力

指定申請そのものは**上位店が支援**します。ただし、申請に必要な資料の多くは**開設者側が準備**する必要があります。上位店の指示に従い、必要な資料を期日までに提出してください。

::: warning 重要
自治体対応や提出判断は**すべて上位店の指示に従って**ください。開設者側が独断で自治体に提出・連絡しないでください。
:::

## 役割分担

| 項目 | 担当 |
|------|------|
| 指定申請書の作成・提出 | 上位店 |
| 自治体との事前相談 | 上位店 |
| 行政対応・問い合わせ | 上位店 |
| 申請に必要な資料の準備 | 開設者（上位店の指示に従う） |
| 物件・職員・備品等の情報提供 | 開設者 |

## 開設者側が準備する可能性がある資料

上位店から依頼があった資料を、期限までに準備・提出してください。

### 法人・事業所情報

- [ ] 法人情報（定款・登記簿謄本等）
- [ ] 事業所名称・所在地
- [ ] 運営体制資料

### 物件関連

- [ ] 物件資料（賃貸借契約書写し等）
- [ ] 平面図
- [ ] 物件内観・外観写真

### 職員関連

- [ ] 職員情報（氏名・資格・経歴等）
- [ ] 雇用契約書
- [ ] 資格証の写し
- [ ] 勤務予定表
- [ ] 組織図 → [雛形を開く](/templates/organization-chart)

### 備品・設備

- [ ] 備品リスト
- [ ] 備品・設備の写真

## 資料準備のチェックリスト

- [ ] 上位店から必要資料リストを受け取った
- [ ] 各資料の提出期限を確認した
- [ ] 法人情報を準備した
- [ ] 物件資料・平面図を準備した
- [ ] 職員情報・雇用契約書・資格証を準備した
- [ ] 勤務予定表を作成した
- [ ] 組織図を作成した
- [ ] 備品リスト・写真を準備した
- [ ] 上位店へ期限内に提出した

## 進め方

1. **リスト受領** — 上位店から必要資料リストと期限を受け取る
2. **準備開始** — 各資料の作成・収集を開始
3. **進捗共有** — 準備状況を上位店に定期的に報告
4. **提出** — 期限までに上位店へ提出
5. **修正対応** — 上位店からの修正依頼があれば速やかに対応
6. **進捗確認** — 申請の進捗は上位店から報告を受ける

## 注意事項

::: danger 必ず守ること
- **自治体への直接連絡・提出は行わない** — すべて上位店経由
- **法的判断が必要な事項は上位店に確認** — 自分で判断しない
- **提出期限を守る** — 遅延は指定取得に影響する
- **資料の正確性を確認** — 虚偽・不備のないよう注意
:::

- 物件契約は指定取得の結果を待ってから行う場合があります → **上位店へ確認**
- 職員の雇用開始日は指定取得時期と調整が必要です → **上位店へ確認**
- 備品リストの内容・数量は上位店の指示に従ってください
