# 就労継続支援B型 開設準備手引書

::: warning 重要な注意事項
この手引書は、就労継続支援B型事業所の開設準備を円滑に進めるための**社内向け資料**です。自治体への指定申請、法的判断、物件の最終可否判断については、**必ず上位店または専門家の指示に従ってください**。
:::

## 手引書の目的

本手引書は、就労継続支援B型事業所を開設する際に、**開設者側が実際に準備すべき内容**を時系列で整理したものです。

自治体への事前相談や指定申請の手続きそのものを解説するものではなく、開設に向けた実務的な準備を進めるためのガイドとしてご利用ください。

## 対象者

- 就労継続支援B型事業所の新規開設を検討・進行している方
- 開設準備の実務を担当するスタッフ
- 開設スケジュールや準備項目の全体像を把握したい方

## この手引書で扱う範囲

| 扱う内容 | 扱わない内容 |
|---------|-------------|
| サビ管・物件・仕事・人員などの開設準備 | 自治体への事前相談の進め方 |
| 開設者側が準備すべき資料・チェックリスト | 指定申請書類の作成手順 |
| 利用者募集・IT環境・開所前準備 | 法的判断・行政判断そのもの |
| テンプレート・実務用フォーマット | 物件の最終可否判断 |

## 上位店と開設者の役割分担

### 上位店が担当すること

- 自治体への事前相談
- 指定申請の支援
- 行政対応
- 物件の最終選定・承認
- 指定取得に関する助言

### 開設者側が担当すること

- サービス管理責任者の確保
- 物件候補の探索
- 物件候補資料の作成
- 上位店への物件候補提出
- 開設資金の準備
- 作業・仕事の確保
- 人員採用
- 備品準備
- IT環境整備
- 利用者募集
- 営業活動
- 開所準備

::: tip 基本方針
**「行政・申請・最終判断は上位店、準備・実行は開設者」** という分担を常に意識して進めてください。判断に迷った場合は、独断せず **上位店へ確認** してください。
:::

## 開設までの全体フロー

1. **開設意思決定** — 開設の意思決定と体制の確認
2. **サビ管確保** — サービス管理責任者の候補確保・面談
3. **物件候補探し** — 条件に合う物件候補の探索
4. **上位店へ物件候補提出** — 物件概要・写真・図面等の提出
5. **上位店による物件確認・選定** — 最終選定・承認（上位店担当）
6. **作業・仕事の確保** — 作業発注元の確保・単価・作業量の確認
7. **資金準備** — 初期費用・運転資金の確保
8. **人員採用** — 管理者・生援・職指等の採用
9. **指定申請準備への協力** — 上位店の指示に従い資料を準備
10. **利用者募集準備** — 見学・体験・営業先への案内準備
11. **IT・集客環境整備** — HP・SNS・問い合わせ導線の整備
12. **開所前最終確認** — 備品・書類・初日の流れ確認
13. **開所** — 事業開始
