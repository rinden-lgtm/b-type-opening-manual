# 開所前最終チェックリスト

開所7日前〜前日に実施する最終確認用チェックリストです。大項目ごとに確認し、未完了項目があれば速やかに対応してください。

::: warning 重要な注意事項
この手引書は、就労継続支援B型事業所の開設準備を円滑に進めるための社内向け資料です。自治体への指定申請、法的判断、物件の最終可否判断については、**必ず上位店または専門家の指示に従ってください**。
:::

::: tip テンプレート版
印刷・記入用の表形式テンプレートは [開所前最終チェックリスト（テンプレート）](/templates/opening-final-checklist) をご利用ください。
:::

## サビ管

- [ ] サービス管理責任者が確定している
- [ ] サビ管研修修了証を確認した
- [ ] 雇用契約書を締結した
- [ ] 勤務開始日が確定している
- [ ] 常勤配置を満たす勤務表になっている → **上位店確認済み**
- [ ] 個別支援計画作成の準備ができている

## 物件

- [ ] 物件の最終選定・承認を上位店から受けている
- [ ] 賃貸借契約を締結した（**上位店確認後**）
- [ ] 内装・レイアウトが完了している
- [ ] 作業室・相談室・事務スペースが確保されている
- [ ] 消防設備・バリアフリー等を確認した

## 仕事

- [ ] メイン作業が決定している
- [ ] 作業発注元が1社以上確保されている
- [ ] 単価・月間作業量・納期を確認した
- [ ] 作業マニュアルを作成した
- [ ] 検品ルールを作成した
- [ ] 10名分の作業量を確保している

## 資金

- [ ] 初期費用の支払いが完了している
- [ ] 運転資金（3〜6ヶ月分）を確保している
- [ ] 開所後の入金タイムラグを考慮している
- [ ] 月次の資金繰り計画がある

## 職員

- [ ] 必要職種の採用が完了している
- [ ] 全職員の雇用契約書を締結した
- [ ] 資格証を確認した
- [ ] 勤務表を作成・共有した
- [ ] 職員研修を実施した
- [ ] 緊急連絡体制を整備した

## 利用者募集

- [ ] 見学・体験の受付体制が整っている
- [ ] パンフレットを準備した
- [ ] 相談支援事業所等へ案内した
- [ ] 問い合わせ対応フローが整っている
- [ ] 見学記録・問い合わせ記録のフォーマットがある

## IT環境

- [ ] ホームページを公開した
- [ ] Googleビジネスプロフィールを登録した
- [ ] LINE公式アカウントを開設した
- [ ] メールアドレス・電話番号が使える
- [ ] 問い合わせフォームが動作する
- [ ] PC・Wi-Fi・プリンターが動作する

## 備品

- [ ] 作業机・椅子を配置した
- [ ] PC・モニターを設置した
- [ ] 事務用品・消耗品を準備した
- [ ] 救急箱等を配置した
- [ ] 掲示物を設置した

## 書類

- [ ] 利用者ファイルのフォーマットがある
- [ ] 記録用紙・帳票を準備した
- [ ] 契約書類のテンプレートがある
- [ ] 個人情報保護等の掲示物がある
- [ ] 指定申請関連資料を上位店に提出済み

## 開所当日準備

- [ ] 全エリアの清掃が完了している
- [ ] 初日スケジュールを全職員に共有した
- [ ] 利用者受入導線を確認した
- [ ] 緊急時対応を全職員が理解している
- [ ] 上位店へ開所報告の連絡先を確認した
