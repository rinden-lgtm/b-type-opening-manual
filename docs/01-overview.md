# 開設準備の全体像

就労継続支援B型事業所の開設には、多くの準備項目が並行して進みます。本章では、開設者が把握すべき全体像と、特に重要なポイントを整理します。

::: warning 重要な注意事項
この手引書は、就労継続支援B型事業所の開設準備を円滑に進めるための社内向け資料です。自治体への指定申請、法的判断、物件の最終可否判断については、**必ず上位店または専門家の指示に従ってください**。
:::

## B型開設に必要な準備項目

| カテゴリ | 主な内容 | 担当 |
|---------|---------|------|
| サビ管 | 候補者の確保・面談・雇用条件確認 | 開設者 |
| 物件 | 候補探索・資料作成・上位店への提出 | 開設者（最終判断は上位店） |
| 仕事 | 作業発注元確保・単価・作業量・マニュアル | 開設者 |
| 資金 | 初期費用・運転資金・物件費・備品費 | 開設者 |
| 人員 | 管理者・生援・職指の採用・勤務体制 | 開設者 |
| 指定申請 | 資料準備への協力 | 開設者（申請は上位店） |
| 利用者募集 | 見学・体験・営業先への案内 | 開設者 |
| IT・集客 | HP・SNS・問い合わせ導線 | 開設者 |
| 開所準備 | 備品・研修・最終確認 | 開設者 |
| 行政対応 | 事前相談・指定申請・助言 | 上位店 |

## 開設者が最初に理解すべき重要ポイント

1. **並行して進める** — サビ管・物件・仕事・人員は同時並行で進める必要があります。順番を待たず、早い段階から動き始めてください。
2. **上位店への確認を怠らない** — 物件契約、行政対応、申請判断は必ず上位店の指示に従います。
3. **仕事（作業）がなければ運営できない** — B型は利用者への作業提供と工賃原資の確保が事業の根幹です。指定取得前から動き始めてください。
4. **資金計画は余裕を持つ** — 開所後の報酬入金にはタイムラグがあります。運転資金は多めに見積もってください。
5. **利用者は開所後すぐには集まらない** — 開所前から見学・体験・営業先への案内を始めてください。

## 特に重要な3項目

以下の3項目は、開設準備の成否を左右する最重要項目です。**最優先で着手**してください。

### 1. サビ管確保

サービス管理責任者（サビ管）は、B型事業所の運営に**法的に必須**の職種です。候補者が見つからなければ、事業所は開設できません。

- 開設意思決定後、**すぐに**候補者の探索を開始
- 研修修了証・実務経験・常勤可否を確認
- [詳細はこちら →](/03-sabikan)

### 2. 物件候補の準備

物件は開設者側が候補を探し、資料を整えて上位店に提出します。**最終選定・承認は上位店**が行います。

- 契約前に**必ず**上位店の確認を受ける
- **独断で契約しない**
- [詳細はこちら →](/04-property)

### 3. 作業・仕事の確保

B型運営では、利用者に提供する作業と工賃原資の確保が**非常に重要**です。作業がなければ利用者を受け入れられません。

- 開所前に最低10名分の作業量を確保
- 将来的に20名規模まで対応できる作業候補も準備
- [詳細はこちら →](/05-work)

## 上位店に任せる部分と自社で動く部分

### 上位店に任せる（開設者は指示に従う）

- 自治体への事前相談・指定申請
- 行政対応全般
- 物件の最終選定・承認
- 指定取得に関する助言・法的判断

### 自社（開設者）で動く

- サビ管候補の探索・面談・雇用調整
- 物件候補の探索・資料作成・提出
- 作業発注元の営業・契約・マニュアル作成
- 資金計画・調達
- 人員採用・勤務体制の構築
- 指定申請に必要な資料の準備（上位店の指示に従う）
- 利用者募集・見学・体験対応
- IT環境・集客導線の整備
- 備品準備・開所前最終確認

::: tip 迷ったら
「これは上位店の管轄か？」と迷ったら、**上位店へ確認**してください。特に物件契約・行政対応・申請判断は、必ず上位店の指示を仰いでください。
:::
