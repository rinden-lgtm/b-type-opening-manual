# サービス管理責任者の確保

サービス管理責任者（以下「サビ管」）は、就労継続支援B型事業所の運営に**法的に必須**の職種です。開設準備の**最優先項目**として、早い段階から確保に着手してください。

::: warning 重要
サビ管が確保できなければ、事業所は開設できません。候補者の探索は開設意思決定後、**すぐに**開始してください。
:::

## サビ管が必要な理由

- 障害福祉サービスの運営において、サービス管理責任者の配置が**法令で義務付け**られています
- 個別支援計画の作成・モニタリング、職員の管理・指導を担います
- 利用者の支援品質と事業所の運営体制の中核となる存在です

## 確保のタイミング

| 時期 | やること |
|------|---------|
| 開設意思決定直後 | 候補者リストの作成・探索開始 |
| 90日前 | 候補者との初回面談 |
| 90日前 | 雇用条件の協議・合意 |
| 30日前 | 雇用契約・勤務開始日の確定 |
| 14日前 | 雇用契約締結・資格証確認 |

## 確認すべき書類

- [ ] サービス管理責任者等基礎研修修了証
- [ ] 相談支援従事者初任者研修（講義部分）修了証
- [ ] サービス管理責任者等実践研修修了証
- [ ] 履歴書
- [ ] 職務経歴書
- [ ] 資格証の写し（該当する場合）
- [ ] 身分証明書

## 面談時の確認項目

| 項目 | 確認内容 |
|------|---------|
| ① サービス管理責任者等基礎研修修了証 | 有無・修了時期 |
| ② 相談支援従事者初任者研修（講義部分）修了証 | 有無・修了時期 |
| ③ サービス管理責任者等実践研修修了証 | 有無・修了時期 |
| 実務経験 | B型・A型・相談支援等の実務経験年数 |
| 常勤勤務 | 常勤勤務が可能か（週5日・1日7時間以上等） |
| 勤務開始日 | いつから勤務開始できるか |
| 雇用条件 | 希望給与・勤務時間・休日等 |
| 兼務 | 他事業所との兼務の有無 |
| 個別支援計画 | 個別支援計画作成の経験 |
| 利用者対応 | 利用者対応・クレーム対応の経験 |

## 注意点

- **兼務の可否**は自治体・上位店の判断が必要な場合があります → **上位店へ確認**
- 研修修了から一定期間経過している場合、追加研修が必要な場合があります → **上位店へ確認**
- サビ管の配置基準（常勤換算等）は自治体により異なります → **上位店へ確認**
- 候補者が見つからない場合は、早めに上位店に相談してください
- 雇用契約書兼労働条件通知書の内容は上位店の指示に従ってください
