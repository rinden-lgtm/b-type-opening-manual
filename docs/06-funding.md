# 開設資金の準備

B型事業所の開設には、初期費用と開所後の運転資金の両方が必要です。余裕を持った資金計画を立ててください。

::: warning 重要
開所後の報酬入金には**1〜2ヶ月程度のタイムラグ**があります。運転資金は多めに見積もってください。
:::

## 資金の内訳

### 初期費用

開設時に一度だけかかる費用です。

| 項目 | 目安金額 | 備考 |
|------|---------|------|
| 物件費用（敷金・礼金） | 50万〜200万円 | 家賃の3〜6ヶ月分程度 |
| 内装費 | 30万〜150万円 | レイアウト変更・工事規模による |
| 備品費 | 30万〜100万円 | PC・家具・事務用品等 |
| 採用費 | 10万〜50万円 | 求人広告・紹介料等 |
| 広告費 | 5万〜30万円 | パンフレット・Web・名刺等 |
| IT環境整備 | 5万〜20万円 | HP制作・ドメイン・SaaS等 |
| その他（登記・許認可関連等） | 10万〜50万円 | 上位店の指示に従う |

**初期費用合計目安：140万〜600万円**

::: tip 幅を持たせる理由
物件の規模・立地、内装の有無、備品の新古、採用方法等により大きく変動します。早い段階で概算を出し、上位店と相談してください。
:::

### 運転資金

開所後、報酬入金までの間に必要な資金です。

| 項目 | 目安（月額） | 備考 |
|------|------------|------|
| 家賃 | 10万〜30万円 | 物件による |
| 人件費 | 80万〜200万円 | 職員数・給与水準による |
| 光熱費・通信費 | 3万〜10万円 | |
| 消耗品・備品補充 | 2万〜5万円 | |
| その他経費 | 5万〜15万円 | |

**運転資金目安：3〜6ヶ月分（300万〜1,500万円程度）**

::: danger 入金タイミングに注意
- 利用者が利用開始しても、報酬の請求・入金には**1〜2ヶ月**かかります
- 開所直後は利用者数が少ないため、収入は立ち上がりに時間がかかります
- **最低3ヶ月分、できれば6ヶ月分**の運転資金を確保してください
:::

## 資金準備のチェックリスト

- [ ] 初期費用の概算を作成した
- [ ] 運転資金（3〜6ヶ月分）を確保した
- [ ] 物件費用（敷金・礼金・前家賃）の資金を確保した
- [ ] 内装費の見積もりを取得した
- [ ] 備品リストと概算費用を作成した
- [ ] 採用費・広告費の予算を確保した
- [ ] 開所後の入金タイムラグを考慮した
- [ ] 資金不足時の対応（借入・追加出資等）を検討した

## 資金計画の進め方

1. **概算作成** — 上記目安を参考に、自社の条件で概算を作成
2. **上位店相談** — 資金計画を上位店と共有し、助言を受ける
3. **調達方法の決定** — 自己資金・借入・出資等
4. **実行計画** — いつ・いくら必要かを時系列で整理
5. **余裕の確保** — 想定の120〜150%を目標に準備

## 注意点

- 内装・備品は見積もりを複数取得し、比較検討してください
- 借入を検討する場合は、返済計画と事業計画の整合性を確認してください → **上位店へ確認**
- 法人設立費用・指定申請関連費用は上位店の指示に従ってください
- 資金計画は定期的に見直し、不足があれば早めに対応してください
